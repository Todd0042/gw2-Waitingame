SHMUP Game for Nexus for while waiting for the event to happen.

Build using AI. If that's a problem for you just don't use it and don't recomment it.

Let me know if you notice problems.

Expected behavior for event detection: random next event by default, if on a map with a specific event it goes by that event.
- world bosses, map metas (partial support).

Controls:
text box available to not leak controls into game. click inside to capture controls, click outside to release.

up, down, left, right - movement

shift, caps lock - fire main weapon or bomb (2 bombs per life)
